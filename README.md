# Creative Image Gallery

## 📌 Description
A modern and responsive image gallery website built using HTML, CSS, and JavaScript.  
The project includes category filtering, hover animations, and a lightbox image preview feature.

## ✨ Features
- Responsive Image Gallery
- Category Filter
- Lightbox Image Preview
- Smooth Hover Effects
- Modern UI Design
- Mobile Friendly Layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
CodeAlpha_Image_Gallery
│
├── gallery.html

├── gallery.js
├── img1.jpg
├── img2.jpg
├── img3.jpg
├── img4.jpg
├── img5.jpg
└── img6.jpg
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open `gallery.html` in your browser.
4. Explore the image gallery.

## 🎯 Project Objective

To learn frontend development concepts including:
- HTML structure
- CSS styling and animations
- JavaScript event handling
- DOM manipulation
